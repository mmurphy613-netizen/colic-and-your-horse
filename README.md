# Colic and Your Horse

This is an instructional design portfolio sample created in Articulate Storyline 360.

The course is designed for horse owners and caretakers and provides an introduction to recognizing signs of colic and understanding appropriate response steps.

## Project Details

- Tool: Articulate Storyline 360
- Format: Interactive eLearning sample
- Audience: Horse owners and caretakers
- Purpose: Support awareness and decision-making related to equine colic
- Design elements: Guided navigation, visual content, learner interaction, and knowledge checks

## Portfolio Note

This project was created as a portfolio sample to demonstrate eLearning design and development using Articulate Storyline 360.
